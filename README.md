# 2026-w2-tools-current_measurement
This respository is a contains the necessary firmware and Python files to get current measurements into a csv. There's definitely better current sensors available but this was so cheap and it gets us pretty close to what we want. Every INA226 module will need to be tuned with a multimeter that can read up to single digit mA's. 

## Hardware

- Arduino UNO
- Arduino UNO hat
- cheap INA226 current measurement sensor module

## Requirements

- INA226 arduino library
- Python with Serial package
